# photoFinder
<h1>AI-Powered Face Recognition Image Search System</h1>
<h2>A solution to quickly find the people you’re looking for among thousands of photos</h2>

<h3>• High-speed image search powered by AI face recognition</h3>
<h3>• Multi-person conditional search (AND / OR)</h3>
<h3>• Highly accurate person detection even in large-scale image datasets</h3>


<h2>👉 The fastest and most accurate way to find people in your photos</h2>
<ul>
  <li>Search all images by folder or file unit</li>
  <li>Automatic person detection using face recognition</li>
  <li>Multi-person conditional search (AND / OR)</li>
  <li>Precision search based on similarity</li>
  <li>Fast processing even with large volumes of images</li>
  <li>Filter photos containing specific individuals only</li>
  <li>Search for photos with multiple people together</li>
</ul>


<h3>1. Overview</h3>
<h4>This program is a face recognition-based image search system designed to quickly find specific individuals within images from user-selected folders or files.</h4>
<h4>Unlike simple filename searches, it analyzes facial data within images to accurately identify and locate the desired person.</h4>


<h3>2. Key Features</h3>

<h4>2.1 Face Recognition-Based Search System</h4>
<ul>
  <li>Automatically detects and analyzes faces in images</li>
  <li>Compares with registered individuals to determine identity</li>
  <li>Provides results based on similarity (%)</li>
  <li>Search by person name</li>
</ul>


<h4>2.2 Multi-Person Conditional Search</h4>
<ul>
  <li>Set multiple individuals as search conditions</li>
  <li>AND condition: Find photos where A and B appear together</li>
  <li>OR condition: Find photos where A or B appears</li>
  <li>Supports complex person combinations</li>
  <li>Accurate result filtering</li>
</ul>


<h4>2.3 Similarity-Based Precision Search</h4>
<ul>
  <li>Comparison based on facial feature vectors</li>
  <li>Results provided as similarity percentage</li>
  <li>Customizable similarity threshold</li>
  <li>Minimizes false positives</li>
</ul>

<h4>2.4 Large-Scale Image Processing</h4>
<ul>
  <li>Handles thousands to tens of thousands of images</li>
  <li>Batch scanning by folder</li>
  <li>Fast indexing and search performance</li>
  <li>Supports common image formats (JPG, PNG, etc.)</li>
</ul>


<h4>2.5 User Convenience Features</h4>
<ul>
  <li>Intuitive UI for person registration and search</li>
  <li>Thumbnail preview of search results</li>
  <li>Filtering and sorting of results</li>
  <li>Save search conditions for repeated use</li>
</ul>


<h3>3. Core Technology</h3>

<h4>✔ AI Face Recognition Algorithm</h4>
<ul>
<li>Automatic face detection in images</li>
<li>Deep learning-based feature extraction and comparison</li>
<li>Robust recognition under varying lighting and angles</li>
</ul>


<h4>✔ Facial Feature Vector Matching</h4>
<ul> 
  <li>Converts faces into numerical data (vectors)</li>
  <li>Calculates similarity against registered individuals</li>
  <li>Fast and accurate matching</li>
</ul>


<h4>✔ Large-Scale Image Indexing</h4>
<ul>
  <li>Pre-analysis and storage of image data</li>
  <li>Instant result retrieval during search</li>
  <li>Optimized performance structure</li>
</ul>
