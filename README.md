# photoFinder
<h1>이미지 얼굴 인식 기반 통합 검색 프로그램</h1>
<h2>수많은 사진 속에서 원하는 사람을 빠르게 찾는 솔루션</h2>
<h3>• AI 얼굴 인식 기반 고속 이미지 검색</h3>
<h3>• 다중 인물 조건 검색 (AND / OR)</h3>
<h3>• 대량 이미지에서도 높은 정확도의 인물 탐지</h3>
<h2>👉 사진 속 인물을 찾는 가장 빠르고 정확한 방법</h2>

<ul>
  <li>폴더 및 파일 단위 이미지 전체 검색</li>
  <li>얼굴 인식 기반 인물 자동 탐지</li>
  <li>다중 인물 조건 검색 (AND / OR)</li>
  <li>유사도 기반 정밀 검색</li>
  <li>대용량 이미지에서도 빠른 처리 속도</li>
  <li>특정 인물이 포함된 사진만 필터링</li>
  <li>여러 인물이 함께 있는 사진 검색 가능</li>
</ul>

<h3>1. 개요</h3>
<h4>본 프로그램은 사용자가 지정한 폴더 또는 파일 내의 이미지에서 특정 인물을 빠르게 검색하기 위한 얼굴 인식 기반 이미지 검색 시스템입니다.</h4>
<h4>단순 파일명 검색이 아닌, 이미지 내 얼굴 데이터를 분석하여 원하는 사람을 정확하게 찾아내는 것이 핵심 기능입니다.</h4>

<h3>2. 주요 기능</h3>
<h4>2.1 얼굴 인식 기반 검색 시스템</h4>
<ul>
  <li>이미지 내 얼굴을 자동으로 탐지 및 분석</li>
  <li>등록된 인물과 비교하여 동일 인물 여부 판단</li>
  <li>유사도(%) 기반 결과 제공</li>
  <li>사람 이름 기준으로 검색 가능</li>
</ul>
<h4>2.2 다중 인물 조건 검색</h4>
<ul>
  <li>여러 인물을 동시에 검색 조건으로 설정 가능</li>
  <li>AND 조건: A와 B가 함께 있는 사진 검색</li>
  <li>OR 조건: A 또는 B가 포함된 사진 검색</li>
  <li>복잡한 인물 조합 검색 지원</li>
  <li>정확한 결과 필터링 가능</li>
</ul>
<h4>2.3 유사도 기반 정밀 검색</h4>
<ul>
  <li>얼굴 특징 벡터 기반 비교</li>
  <li>유사도 퍼센트로 결과 제공</li>
  <li>유사도 기준값 설정 가능</li>
  <li>오탐지 최소화</li>
</ul>
<h4>2.4 대용량 이미지 처리</h4>
<ul>
  <li>수천~수만 장 이미지 처리 가능</li>
  <li>폴더 단위 일괄 스캔</li>
  <li>빠른 인덱싱 및 검색 속도</li>
  <li>이미지 포맷 자동 처리 (JPG, PNG 등)</li>
</ul>
<h4>2.5 사용자 편의 기능</h4>
<ul>
  <li>직관적인 UI 기반 인물 등록 및 검색</li>
  <li>검색 결과 썸네일 미리보기 제공</li>
  <li>검색 결과 필터링 및 정렬 기능</li>
  <li>반복 검색을 위한 조건 저장</li>
</ul>

<h3>3. 핵심 기술 특징</h3>
<h4>✔ 얼굴 인식 AI 알고리즘</h4>
<ul>
<li>이미지에서 얼굴 영역 자동 검출</li>
<li>딥러닝 기반 특징 추출 및 비교</li>
<li>조명, 각도 변화에도 강건한 인식 성능</li>
</ul>

<h4>✔ 얼굴 특징 벡터 매칭</h4>
<ul> 
  <li>얼굴을 수치 데이터(벡터)로 변환</li>
  <li>등록 인물과 비교하여 유사도 계산</li>
  <li>빠르고 정확한 매칭 수행</li>
</ul>

<h4>✔ 대용량 이미지 인덱싱</h4>
<ul>
  <li>이미지 데이터 사전 분석 및 저장</li>
  <li>검색 시 즉시 결과 반환</li>
  <li>속도 최적화 구조 적용</li>
</ul>
